# Cybersecurity Certifications Reference

A comprehensive reference for cybersecurity certifications — organized by domain, with cost, difficulty, renewal requirements, and who should pursue each. For career-progression roadmaps, see [Career Paths & Cert Roadmap](CAREER_PATHS.md).

---

## Quick Selection Guide

| Goal | Recommended Cert |
|---|---|
| First security cert (IT background) | CompTIA Security+ |
| First security cert (no IT background) | CompTIA Security+ or CC (ISC2) |
| SOC analyst role | CompTIA CySA+, BTL1, GCIH |
| Penetration tester (entry) | eJPT, CompTIA PenTest+ |
| Penetration tester (serious) | OSCP (OffSec) |
| Red team operator | CRTO, CRTE, OSED |
| Cloud security | AWS Security Specialty, CCSP, GCP PCSE |
| Detection engineering | GCIA, GCFE, vendor SIEM certs |
| Incident response | GCIH, GCFE, GCFA |
| Digital forensics | GCFE, GCFA, EnCE, CCE |
| Threat intelligence | GCTI, CTIA |
| Management / CISO track | CISSP, CISM, CRISC |
| Risk quantification | CRISC, FAIR-P |
| AppSec / DevSecOps | GWEB, GWAP, CSSLP |
| GRC / compliance | CISM, CRISC, CISA, ISO 27001 LA |
| Cloud architecture | CCSP, SABSA, AWS SA Pro |

## Entry-Level Certifications

### CompTIA Security+
| Attribute | Detail |
|---|---|
| Issuer | CompTIA |
| Cost | ~$392 (exam voucher) |
| Experience required | None (2 years IT recommended) |
| Exam format | 90 questions, multiple choice + performance-based, 90 min |
| Passing score | 750/900 |
| Renewal | CE: 50 CPEs in 3 years, or retake |
| DoD 8570 | Yes (IAT Level II, IAM Level I) |
| Best for | Entry-level security analyst, junior SOC, IT admin moving into security |

### ISC2 Certified in Cybersecurity (CC)
| Attribute | Detail |
|---|---|
| Issuer | ISC2 |
| Cost | Free exam (limited time offer / One Million Certified program) |
| Experience required | None |
| Exam format | 100 questions, 2 hours |
| Renewal | 45 CPEs in 3 years, annual AMF |
| Best for | True career changers with no security background; free entry point |

### CompTIA Network+
| Attribute | Detail |
|---|---|
| Issuer | CompTIA |
| Cost | ~$338 |
| Focus | Networking fundamentals — prerequisite knowledge for security |
| Best for | Before Security+ if no networking background |

### CompTIA A+
| Attribute | Detail |
|---|---|
| Issuer | CompTIA |
| Cost | ~$246 per exam (2 exams required) |
| Focus | IT fundamentals, hardware, OS, troubleshooting |
| Best for | Absolute beginners building foundational IT literacy |

## Intermediate Certifications

### GIAC Security Essentials (GSEC)
| Attribute | Detail |
|---|---|
| Issuer | GIAC (SANS) |
| Cost | ~$949 (exam only) / $8,000+ with SANS course |
| Experience required | None formally, but substantial knowledge expected |
| Exam format | Open book, 180 questions, 5 hours |
| Renewal | 36 CPEs in 4 years |
| Best for | Broad security knowledge validation; strong alternative to Security+ |

### CompTIA CySA+ (Cybersecurity Analyst)
| Attribute | Detail |
|---|---|
| Issuer | CompTIA |
| Cost | ~$392 |
| Focus | Threat detection, SIEM, behavioral analysis, incident response |
| DoD 8570 | Yes (IAT Level III, CSSP Analyst) |
| Best for | SOC analyst L2/L3, detection engineering entry |

### eLearnSecurity Junior Penetration Tester (eJPT)
| Attribute | Detail |
|---|---|
| Issuer | INE Security (formerly eLearnSecurity) |
| Cost | $200 |
| Format | Practical exam (72-hour lab) |
| Best for | First offensive cert; beginner-friendly practical pentesting |

### CompTIA PenTest+
| Attribute | Detail |
|---|---|
| Issuer | CompTIA |
| Cost | ~$392 |
| Focus | Pentest methodology, tools, reporting |
| DoD 8570 | Yes (CSSP Analyst, CSSP Infrastructure Support) |
| Best for | Entry-to-mid pentest certification alongside eJPT |

### Certified Ethical Hacker (CEH)
| Attribute | Detail |
|---|---|
| Issuer | EC-Council |
| Cost | ~$1,199 (exam only) |
| Focus | Broad ethical hacking domains |
| DoD 8570 | Yes (IAT Level II, CSSP Analyst) |
| Note | More recognized in enterprise/government than in technical pentest community |
| Best for | Government contractors, those needing DoD 8570 compliance |

## Offensive Security / Red Team

### OffSec Certified Professional (OSCP)
| Attribute | Detail |
|---|---|
| Issuer | Offensive Security |
| Cost | $1,499 (90-day lab access + exam) |
| Format | 24-hour hands-on exam — compromise 3 standalone machines + Active Directory set |
| Difficulty | High; industry gold standard |
| Renewal | None (lifetime) |
| Best for | Penetration testers; highly respected by employers |

### OffSec Experienced Penetration Tester (OSEP)
| Attribute | Detail |
|---|---|
| Issuer | Offensive Security |
| Cost | $1,499 |
| Focus | Evasion techniques, antivirus bypass, advanced Windows exploitation |
| Prerequisite | OSCP recommended |
| Best for | Mid-level pentesters expanding beyond basic OSCP skills |

### Certified Red Team Professional (CRTP)
| Attribute | Detail |
|---|---|
| Issuer | Altered Security (Nikhil Mittal) |
| Cost | ~$249 (bootcamp + exam) |
| Focus | Active Directory attacks and defense, Kerberos attacks |
| Format | 24-hour practical AD lab exam |
| Best for | Red teamers focused on Windows/AD environments |

### Certified Red Team Expert (CRTE)
| Attribute | Detail |
|---|---|
| Issuer | Altered Security |
| Cost | ~$400 |
| Focus | Advanced AD, Azure AD, cross-forest attacks |
| Prerequisite | CRTP recommended |
| Best for | Advanced AD red team operators |

### Certified Red Team Operator (CRTO)
| Attribute | Detail |
|---|---|
| Issuer | Zero-Point Security (RastaMouse) |
| Cost | ~£399 |
| Focus | Cobalt Strike, C2 operations, operational security, OPSEC |
| Format | 48-hour practical exam using Snap Labs |
| Best for | Red team operators learning C2 operations and OPSEC |

### OffSec Web Expert (OSWE)
| Attribute | Detail |
|---|---|
| Issuer | Offensive Security |
| Cost | $1,499 |
| Focus | Source code review, white-box web app exploitation |
| Best for | Application security penetration testers |

### OffSec Exploit Developer (OSED)
| Attribute | Detail |
|---|---|
| Issuer | Offensive Security |
| Cost | $1,499 |
| Focus | Windows exploit development, DEP/ASLR bypass, SEH overwrites |
| Best for | Exploit developers, advanced red teamers |

## Certification Progression Map

```
Entry (0–2 years)
├── CompTIA Security+
├── ISC2 CC (free)
└── BTL1 (practical blue team)

Mid-Level (2–5 years)
├── Blue Team: CySA+, GCIH, GCFE
├── Offensive: eJPT → OSCP → CRTO/CRTE
├── Cloud: AWS/Azure/GCP security specialty
├── Forensics: GCFE → GCFA
└── Intel: GCTI, CTIA

Senior / Specialist (5+ years)
├── Management: CISSP, CISM, CRISC
├── AppSec: CSSLP, GWAPT, OSWE
├── ICS/OT: GICSP, GRID
├── Cloud Architecture: CCSP
├── K8s: CKA → CKS
└── Advanced Red Team: OSEP, OSED, CRTE
```

## Cost Comparison

| Tier | Certifications | Typical Cost Range |
|---|---|---|
| Free / Low cost | ISC2 CC, BTL1, eJPT, PortSwigger | Free – $250 |
| Mid-range | Security+, CySA+, CEH, SC-200, AZ-500 | $165 – $750 |
| Premium | CISSP, CISM, OSCP, CCSP, AWS Security | $300 – $1,500 |
| High (with training) | GIAC certs with SANS courses | $7,000 – $9,000 |

> SANS frequently offers Work Study programs (free if you help with event logistics) and OnDemand bundles. Many employers reimburse for GIAC exams taken after SANS courses.
