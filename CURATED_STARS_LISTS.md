## 1. Daily Operational Toolkit
Tier-one tooling for day-to-day security work across reconnaissance, web testing, detection, forensics, and supply-chain scanning. Curated for practitioners who need to move fast across disciplines without context-switching toolkits.

## 2. Active Directory Offensive Operations
End-to-end Active Directory and Entra ID attack tooling covering enumeration, credential abuse, ADCS exploitation, Kerberos relay paths, NTLM coercion, and domain-trust escalation. Includes BloodHound, Mimikatz, Impacket, NetExec, Certipy, ROADtools, Coercer, Inveigh, and the GhostPack suite.

## 3. Command-and-Control and Post-Exploitation Frameworks
Adversary-emulation backbones spanning open-source C2 platforms (Metasploit, Sliver, Havoc, Mythic, Empire, NimPlant, Caldera) and tradecraft primitives (Donut, sRDI, COFF loaders) for red-team operations and detection validation.

## 4. Malware Analysis and Reverse Engineering
Static, dynamic, and automated malware-triage workbench. Covers disassemblers and debuggers (Ghidra, x64dbg, radare2, ImHex), capability detection (capa, FLARE-FLOSS), sandboxing (CAPEv2), unpacking (PE-sieve), pattern matching (YARA), memory forensics (Volatility), and dynamic instrumentation (Frida).

## 5. Digital Forensics and Incident Response
Field kit for live response, host triage, memory and disk forensics, timeline reconstruction, and ticket-driven IR. Includes Velociraptor, Volatility, Plaso, Sleuthkit, Autopsy, the Eric Zimmerman tools, KAPE, TheHive, Cortex, LogonTracer, DeepBlueCLI, and Hayabusa.

## 6. Detection Engineering and Analytics
Authoring, conversion, and validation pipeline for SIEM and EDR detections. Sigma plus pySigma backends, Elastic detection-rules, Splunk security_content, Microsoft 365 Defender hunting queries, MSTICPy, Sentinel content, signature-base, and Atomic Red Team for behavioural validation.

## 7. MITRE ATT&CK Framework and CTID Tooling
Authoritative ATT&CK data, mapping libraries, and Center for Threat-Informed Defense projects. Includes ATT&CK Navigator, attack-stix-data, mitreattack-python, the Cyber Analytics Repository (CAR), Attack-Flow, Mappings Explorer, Adversary Emulation Library, Sensor Mappings, TRAM, Caldera, and Atomic Red Team.

## 8. Cyber Threat Intelligence Platforms and Feeds
CTI platforms, STIX/TAXII libraries, and curated indicator sources. MISP, OpenCTI, Yeti, oasis-open STIX2 and TAXII clients, CrowdStrike MISP-tools, Azure MDTI Solutions, Maltrail, IPSum, FireHOL blocklists, and Phishing.Database.

## 9. Bug Bounty and Web Application Reconnaissance
Recon-to-report workflow for bug bounty and web pentests. Project Discovery suite (nuclei, subfinder, httpx, katana, naabu, dnsx), Amass, ffuf, gobuster, dirsearch, aquatone, EyeWitness, SSRFmap, GraphQLmap, reconftw, HowToHunt, can-i-take-over-xyz, keyhacks, and HUNT.

## 10. Cloud and Container Security: Attack and Defense
AWS, Azure, GCP, and Kubernetes security tooling for both offensive enumeration and defensive posture management. CloudMapper, ScoutSuite, Prowler, cloud_enum, Policy Sentry, Checkov, Trivy, kube-bench, kube-hunter, Tracee, Falco, kube-linter, badPods, Grype, and Syft.

## 11. OSINT and External Reconnaissance
Open-source intelligence collection across people, infrastructure, accounts, and breach data. Sherlock, Maigret, theHarvester, Holehe, GHunt, Spiderfoot, social-analyzer, FinalRecon, twint, webscreenshot, and dnsvalidator.

## 12. AI and LLM Security Engineering
Red-teaming, prompt-injection assessment, and runtime defence for LLM-backed systems. Microsoft PyRIT, NVIDIA Garak, Protect AI llm-guard and Rebuff, PentestGPT, Strix, and the Anthropic Cybersecurity Skills collection.

## 13. Enterprise Security Vendor Tooling
Official repositories from major security vendors. CrowdStrike (MISP-tools, falconpy, Falcon-Toolkit, falcon-installer), Microsoft (MSRC research, MSTICPy, PyRIT, Security-101, PSRule for Azure), Azure Sentinel, Microsoft Defender Threat Intelligence, defender-docs, Cisco Talos IOCs, and Palo Alto Unit 42.

## 14. Cybersecurity Learning Resources and Catalogs
Reference catalogs and curated awesome-lists across hacking, detection engineering, threat detection, incident response, threat intelligence, malware analysis, forensics, reversing, AppSec, blue team, and pentest. Starting points for any deep dive.

## 15. TeamStarWolf - Production Stack
TeamStarWolf-authored projects (LimeWire, ATTACK-Navi, PokeNav) plus the upstream tooling they rely on for development and packaging.

## 16. Vulnerability and Attack Surface Management
Open-source vulnerability management platforms, vulnerability scanners, attack surface management, and external exposure monitoring. DefectDojo, archerysec, Greenbone OpenVAS/gvmd, Wiz Open-CVDB, OWASP ZAP, Coraza WAF, dnstwist, ThreatMapper, SecretScanner, rEngine, Sn1per, surf, scan4all, shellhub, Brinqa LDAP-CLI, and the cisagov vulnrichment + CVE Project automation working group.

## 17. Enterprise EDR and Endpoint Security Vendors
Official tooling and integrations from major endpoint security vendors. **CrowdStrike Falcon** (falconpy, falcon-helm, falcon-operator, foundry samples, caracara, container-image-scan, gofalcon, falcon-integration-gateway). **Microsoft Defender for Endpoint** (MDE PowerBI templates, MSTIC-Sysmon, MSTICPy, MDTI Solutions, defender-docs, M365 Defender hunting queries, Cloud-Katana, counterfit, SimuLand). **Tanium** (Tanium-OSS, Sentinel integration). **Armis** (CLI, SDK, BlueBorne research, ICS Security Tools, Kenna toolkit).

## 18. SOAR and Security Automation
Security orchestration, automation, and response platforms plus SIEM-adjacent automation. Demisto/Cortex XSOAR content, demisto-sdk, Shuffle and Shuffle python-apps, Splunk Ansible, Splunk security_content, Elastic integrations and Kibana. For building SOC playbook automation and detection content pipelines.

## 19. Compliance, GRC and OSCAL

## 20. Identity, Zero Trust and Privileged Access

## 21. All-in-One Hacker Toolkit and Recon
Practitioner-grade omnibus list combining hacking-book companion code (Black Hat Python, Violent Python, Practical Malware Analysis, PacktPublishing pentesting and reverse engineering), end-to-end recon and pentest frameworks (Sn1per, Recon-ng + marketplace, AutoRecon, Bluto, Discover, Sublist3r, Photon, ParamSpider, FavFreak, hackingtool, Coldfire), foundational exploitation suites (BeEF, Responder, sqlmap, Wapiti, ZAP community scripts), and the most-referenced hacking knowledge bases (h4cker, RedTeam-Tools, Awesome-CTF, ctf-tools, Free-Security-eBooks, Penetration-Testing, Infosec_Reference, hackerschoice tips, jhaddix tbhm, Ignite cheatsheets). The single list to open when you need to do everything from external recon to post-exploitation in one place.
