# TeamStarWolf — Curated GitHub Stars Lists

> Maintained by [**@WolfenLabs**](https://x.com/WolfenLabs)

Instead of dumping entire categories into Lists, these are **15 hand-picked toolkits** — each List answers the question *"what would I open if I needed to do X today?"*

The companion script [`create_starred_lists.py`](create_starred_lists.py) creates all of these in your GitHub Stars Lists in one run (after `gh auth refresh -s user`).
