## Projects

### [ATTACK-Navi](https://github.com/TeamStarWolf/ATTACK-Navi)

[![Deploy to GitHub Pages](https://github.com/TeamStarWolf/ATTACK-Navi/workflows/Deploy%20to%20GitHub%20Pages/badge.svg)](https://github.com/TeamStarWolf/ATTACK-Navi/actions/workflows/deploy.yml)
[![Docker Build](https://github.com/TeamStarWolf/ATTACK-Navi/workflows/Docker%20Build/badge.svg)](https://github.com/TeamStarWolf/ATTACK-Navi/actions/workflows/docker.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://github.com/TeamStarWolf/ATTACK-Navi/blob/main/LICENSE)

MITRE ATT&CK workbench for coverage review, detection engineering, exposure mapping, and threat-intelligence correlation.

**Capabilities**
- 24 heatmap modes across coverage, detection, exposure, compliance, and risk
- 4.5M CVE mappings; live integrations with MISP, OpenCTI, EPSS, CISA KEV, NVD, Elastic, Splunk, Sigma, Atomic Red Team, ExploitDB, and Nuclei
- STIX 2.1 import/export, custom technique editing, and collection sharing
- Deployable via Docker or GitHub Pages

[Repository](https://github.com/TeamStarWolf/ATTACK-Navi) | [Live Site](https://teamstarwolf.github.io/ATTACK-Navi/) | [Docs](https://github.com/TeamStarWolf/ATTACK-Navi/blob/main/docs/README.md)

### [LimeWire](https://github.com/TeamStarWolf/LimeWire)

[![CI](https://github.com/TeamStarWolf/LimeWire/workflows/CI/badge.svg)](https://github.com/TeamStarWolf/LimeWire/actions/workflows/check.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://github.com/TeamStarWolf/LimeWire/blob/main/LICENSE)

Python desktop audio studio with 25 tabs covering download, analysis, editing, effects, stem separation, DJ mixing, playlist transfer, and batch processing.

**Capabilities**
- Cross-service integrations: Spotify, YouTube, TIDAL, SoundCloud, Deezer, Apple Music, and Amazon Music (OAuth 2.0 PKCE)
- Demucs stem separation, VST3/AU plugin hosting via pedalboard
- Sandboxed plugin system with DPAPI token encryption at rest
- Ships as a standalone executable for Windows

[Repository](https://github.com/TeamStarWolf/LimeWire) | [Latest Release](https://github.com/TeamStarWolf/LimeWire/releases/latest)

### [PokeNav](https://github.com/TeamStarWolf/PokeNav)

[![Deploy to GitHub Pages](https://github.com/TeamStarWolf/PokeNav/workflows/Deploy%20to%20GitHub%20Pages/badge.svg)](https://github.com/TeamStarWolf/PokeNav/actions/workflows/deploy.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://github.com/TeamStarWolf/PokeNav/blob/main/LICENSE)

Offline-first Pokemon encyclopedia with game-aware browsing, trainer archives, and linked reference data.

**Capabilities**
- Browsing scoped to any main-series game version
- Full trainer battle archive with team compositions and move sets
- Cross-linked species, moves, abilities, items, regions, and locations
- Static deployment via GitHub Pages; no backend required
