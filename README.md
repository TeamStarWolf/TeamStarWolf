## What I Like Building

- polished tools that feel like products, not demos
- data-heavy interfaces that stay readable and fast
- local-first and static-first applications that are easy to ship
- security-aware workflows, documentation, and release hygiene

## Current Snapshot

- `LimeWire` is the most productized repo right now, with a full release, wiki, and stronger desktop-tooling baseline.
- `PokeNav` is focused on becoming a trustworthy, browse-first encyclopedia instead of a shallow card-grid Pokedex clone.
- `ATTACK-Navi` is the most ambitious analysis surface, combining ATT&CK navigation, threat intelligence, and exposure workflows in one interface.

## Profile Goals

This profile is organized around a small set of projects that each show a different strength:

- desktop product engineering
- rich frontend UX for structured data
- cybersecurity analysis tooling

More public repos and refinements will continue to land here as those projects mature.
