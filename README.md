## Main Project

### [ATTACK-Navi](https://github.com/TeamStarWolf/ATTACK-Navi)

ATTACK-Navi is a browser-based cybersecurity analysis workbench built around the MITRE ATT&CK framework.

It is designed to help analysts and defenders explore techniques, review control and detection coverage, connect threat intelligence, and work with mapped security data in one place.

Useful entry points:

- [Repository](https://github.com/TeamStarWolf/ATTACK-Navi)
- [Live Site](https://teamstarwolf.github.io/ATTACK-Navi/)
- [Documentation](https://github.com/TeamStarWolf/ATTACK-Navi/blob/main/docs/README.md)
- [Security Policy](https://github.com/TeamStarWolf/ATTACK-Navi/blob/main/SECURITY.md)

What it is useful for:

- exploring ATT&CK beyond a static matrix
- reviewing coverage and gaps across detections, controls, and mitigations
- correlating techniques with sources such as Sigma, YARA, MISP, OpenCTI, and CISA KEV
- exporting analysis into reports and other shareable artifacts
