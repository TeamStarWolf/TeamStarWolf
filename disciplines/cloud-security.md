# Cloud Security

Securing cloud infrastructure, containers, identity, and IaC across AWS, Azure, GCP, and Kubernetes environments.

---

## Tools & Repositories

- [prowler](https://github.com/prowler-cloud/prowler) — Cloud security posture assessment (AWS/Azure/GCP)
- [pacu](https://github.com/RhinoSecurityLabs/pacu) — AWS exploitation framework
- [ScoutSuite](https://github.com/nccgroup/ScoutSuite) — Multi-cloud security auditing tool
- [cloudmapper](https://github.com/duo-labs/cloudmapper) — AWS environment analysis and visualization
- [AADInternals](https://github.com/Gerenios/AADInternals) — Azure AD / M365 offensive and defensive toolkit
- [ROADtools](https://github.com/dirkjanm/ROADtools) — Azure AD recon and attack framework
- [checkov](https://github.com/bridgecrewio/checkov) — IaC security scanning
- [trivy](https://github.com/aquasecurity/trivy) — Container and cloud vulnerability scanner
- [falco](https://github.com/falcosecurity/falco) — Runtime container security monitoring
- [untitledgoosetool](https://github.com/cisagov/untitledgoosetool) — CISA Azure/M365 incident response tool
