# Detection Engineering

Building, testing, and tuning detections across log sources, SIEMs, and EDR platforms to reliably identify adversary behavior.

---

## Tools & Repositories

- [sigma](https://github.com/SigmaHQ/sigma) — Generic SIEM detection rule format
- [pySigma](https://github.com/SigmaHQ/pySigma) — Convert Sigma rules to any SIEM query language
- [sysmon-config](https://github.com/SwiftOnSecurity/sysmon-config) — SwiftOnSecurity Sysmon configuration
- [sysmon-modular](https://github.com/olafhartong/sysmon-modular) — Modular Sysmon configuration
- [osquery](https://github.com/osquery/osquery) — SQL-powered OS instrumentation for detection
- [elastic/detection-rules](https://github.com/elastic/detection-rules) — Elastic SIEM detection rules
- [splunk/security_content](https://github.com/splunk/security_content) — Splunk Security detection content
- [hayabusa](https://github.com/Yamato-Security/hayabusa) — Sigma-based Windows event log threat hunting
- [DeepBlueCLI](https://github.com/sans-blue-team/DeepBlueCLI) — PowerShell event log threat hunting
- [velociraptor](https://github.com/Velocidex/velociraptor) — Live endpoint forensics and detection response
