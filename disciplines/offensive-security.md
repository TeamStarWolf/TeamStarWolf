# Offensive Security

Penetration testing, red teaming, and adversary emulation to identify and validate security weaknesses before attackers do.

---

## Tools & Repositories

- [metasploit-framework](https://github.com/rapid7/metasploit-framework) — The standard exploitation framework
- [BloodHound](https://github.com/SpecterOps/BloodHound) — Active Directory attack path mapping
- [impacket](https://github.com/fortra/impacket) — Python library for Windows/AD network protocols
- [NetExec](https://github.com/Pennyw0rth/NetExec) — Network pentesting Swiss army knife (CrackMapExec successor)
- [Sliver](https://github.com/BishopFox/sliver) — Open source adversary emulation framework
- [Havoc](https://github.com/HavocFramework/Havoc) — Modern C2 framework
- [Rubeus](https://github.com/GhostPack/Rubeus) — Kerberos abuse toolkit
- [PEASS-ng](https://github.com/peass-ng/PEASS-ng) — LinPEAS/WinPEAS privilege escalation scripts
- [Responder](https://github.com/lgandx/Responder) — LLMNR/NBT-NS poisoner
- [LOLBAS](https://github.com/LOLBAS-Project/LOLBAS) — Living Off The Land Binaries (Windows)
